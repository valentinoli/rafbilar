**Setjið upp alla npm pakkana sem skilgreindir eru í package.json**
* npm install

**Keyrið eslint-villuathugun fyrir javascript**
* npm run eslint -s

**Keyrið stylelint-villuathugun fyrir CSS**
* npm run stylelint -s

**Kveikið loks á vefþjón**
* npm start
